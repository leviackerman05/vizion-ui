# VIZION_UI

An AI-powered frontend that lets users describe ideas in plain text and generate Manim-based animations automatically. This React-based interface interacts with a FastAPI backend and provides a seamless visual scripting experience.

---

## 🚀 Features

- ✍️ Natural language prompt input
- 🎥 Animated video preview panel
- 💬 Chat-based session flow
- ⚙️ Model-aware prompt engine (Gemini Flash/Pro)
- 🪄 Auto video rendering with Manim backend
- 🔐 Firebase Auth (Login/Signup/Session tracking)
- 📦 Clean UI built with Tailwind CSS + shadcn/ui

---

## 🛠️ Tech Stack

- **React** (Vite + TypeScript)
- **Tailwind CSS** for styling
- **shadcn/ui** for UI components
- **Firebase Authentication**
- **Manim** (via backend) for animation rendering

---

## 📦 Setup Instructions

### Prerequisites

- Node.js (v18+ recommended)
- npm or yarn

### Getting Started

```bash
# Clone the repository
git clone <your-repo-url>
cd vizion_ui

# Install dependencies
npm install

# Start the development server
npm run dev
```
